# 🤟 Libras System - Frontend

Sistema de gestão de ensino de Libras desenvolvido com **Vue.js 3** e **Vite**.

## 🚀 Funcionalidades
- **Autenticação JWT**: Login seguro com controle de sessão.
- **Níveis de Acesso (RBAC)**: Dashboards distintos para Administradores e Alunos.
- **Interface Responsiva**: Design moderno focado na experiência do usuário.

## 🛠️ Tecnologias Utilizadas
- [Vue.js 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Axios](https://axios-http.com/)
- [Vue Router](https://router.vuejs.org/)
